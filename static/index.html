Desktop Utility

输入一条 X / Twitter 链接，自动生成详情页截图

截图会直接保存到当前工具文件夹里的 screenshots/ 目录。 如果帖子正文不是中文，可以先点“获取翻译”单独检查正文和中文内容。 确认无误后再点“开始截图”，就会把翻译一并带进最终图片；如果一开始直接点截图，则只生成原图。 如果某些推文需要登录才能访问，可以勾选“显示浏览器”，在打开的浏览器里完成登录后再截图。

直接点“开始截图”会生成原图。先点“获取翻译”，确认内容后再点“开始截图”，则会把翻译一并带进最终图片。

如果推文里有视频，可以直接填想截取的时间点。比如填 `2` 表示第 2 秒，填 `10.5` 表示第 10.5 秒，也可以用 `01:23` 这样的分:秒格式。留空也没关系，工具会自动帮你尽量避开开头的封面帧或黑帧。

翻译确认

点击“获取翻译”后，这里会显示逐段原文和可编辑中文。随后点击“开始截图”会把这里的内容带进最终图片。

使用说明

  • 支持 `x.com`、`twitter.com`、`mobile.twitter.com` 的单条推文链接。
  • 默认优先截取推文详情页；如果详情页无法加载，会自动回退到嵌入卡片截图。
  • 直接点击“开始截图”会生成原图；先点“获取翻译”再截图，会把确认后的中文一并放进图片。
  • 翻译确认区里可以逐段修改中文，也可以一键复制原文去第三方翻译；作者名、时间等信息不会被翻译。
  • 如果推文里有视频，可以直接填时间点，比如 `2`、`10.5`，或者 `01:23`。
  • 截图文件会带上推文 ID 和时间戳，方便反复保存不覆盖。
